# 🌸 Outlet das Marias

Loja online de perfumes com sistema completo de vendas.

## 🛍️ Funcionalidades

- Catálogo de perfumes
- Filtro por categorias (Árabes, Cítricos, Amadeirados, etc)
- Busca de produtos
- Painel administrativo
- Edição e exclusão de produtos
- Venda direta via WhatsApp

## 💻 Tecnologias

- HTML
- CSS
- JavaScript
- Firebase
